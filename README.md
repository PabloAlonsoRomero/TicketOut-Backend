## 💻 Descripción General del Sistema
TicketsOut es un sistema web diseñado para la gestión de tickets de incidencias dentro de una empresa

El backend proporciona una API REST encargada de:

- 👤 Registro y autenticación de usuarios
- 🎫 Creación y administración de tickets de issues
- 🔄 Actualización del estado de los tickets
- 🗂 Organización y consulta de incidencias registradas

Cada ticket podrá contener información como:
- Título del issue
- Descripción del problema
- Prioridad
- Estado (Abierto, En proceso, Resuelto, Cerrado)
- Usuario que lo creó
- Fecha de creación

El sistema busca mejorar la organización, trazabilidad y control de problemas internos dentro de una empresa
